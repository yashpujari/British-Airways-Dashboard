**Overview**
This Tableau dashboard provides an interactive analysis of British Airways customer reviews collected between 2016 and 2023. It highlights overall performance trends and customer satisfaction across various service categories, aircraft types, and regions.

**Key Features** 
1) Dynamic Filters: Filter data by seat type, traveler type, aircraft, and time period.

2) KPI Overview: Displays average ratings for key service metrics such as cabin staff, entertainment, food, ground service, seat comfort, and value for money.

3) Trend Analysis: Visualizes monthly average ratings to identify performance changes over time.

4) Geographic Insights: World map showing average customer ratings by country.

5) Aircraft Comparison: Compares average ratings and review counts across different aircraft models.

To identify patterns in passenger satisfaction and uncover actionable insights to help improve British Airwaysâ€™ service quality and customer experience.
